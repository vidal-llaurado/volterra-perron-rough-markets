# Volterra-Perron Rough Markets

**Latent Contagion, Compression, and Option-Manifold Pricing**

This repository contains the public source package for a sequence of papers on directed rough-volatility contagion, dynamic observability, source-screened inference, Perron synchronization, risk-neutral compression, residual pricing, listed-option relative value, and option-manifold recovery.
